Resource Attributions

The following external resources were utilized to build the RheignCred Orthopedic Hospital digital experience.

Images & Media

  • Specialist Portraits: All doctor images were sourced from Unsplash and optimized to WebP format for performance.
  • Hero Hospital Image: Sourced via Pexels.
  • Hospital Logo: Medicine icon adapted from Google Material Symbols.

Typography & Icons

  • Fonts: 'Montserrat' (Headings) and 'Open Sans' (Body) provided by Google Fonts.
  • Interface Icons: Hamburger and Close icons synthesized from standard UTF-8 characters.

Code & Technical Logic

  • Form Validation: Hidden timestamp and email validation patterns inspired by MDN Web Docs.
  • Modular JavaScript: ES Module structure (import/export) follows WDD231 curriculum standards.
  • Responsive Grid: The auto-fit and minmax layout logic adapted from CSS-Tricks.
